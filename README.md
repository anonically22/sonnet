# Sonnet

A dark VS Code color theme extension shipping **two distinct variants** — each with its own aesthetic identity. More variants will be added under the same Sonnet name over time.

## Variants

### Sonnet Crimson

Pure black base, near-white foreground, blood-red and dark crimson accents. Sharp, high-contrast, severe — like ink on black paper.

## How to switch

Open the **Color Theme** picker with `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on macOS), then select **Sonnet Crimson** or **Sonnet Tsar**.

## Installation

### From the Marketplace

1. Open the Extensions view (`Ctrl+Shift+X`) and search for **Sonnet**.
2. Click **Install**.
3. Open **Color Theme** (`Ctrl+K Ctrl+T`) and choose **Sonnet Crimson** or **Sonnet Tsar**.

*(Not yet published — install via `.vsix` for now, see below.)*

### From VSIX (local install)

1. Download the latest `sonnet-*.vsix` release.
2. Open VS Code and run **Extensions: Install from VSIX…** from the Command Palette (`Ctrl+Shift+P`).
3. Select the downloaded `.vsix` file.
4. Open **Color Theme** (`Ctrl+K Ctrl+T`) and choose your preferred variant.

**Antigravity:** Antigravity keeps its own extension registry, separate from VS Code, so the VSIX install has to be done from inside Antigravity itself — open the Extensions panel, use the `...` menu → **Install from VSIX...**, then select the file and reload when prompted.
